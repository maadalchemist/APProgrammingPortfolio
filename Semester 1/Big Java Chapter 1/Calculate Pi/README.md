# Calculate Pi

This console app estimates pi

## Psudocode

### Problem Statement
* Calculate pi

### Inputs and Outputs
* double pi
* int subtract

### Tasks
* set pi to 1
* for 2,147,483,647 times (max int size)
  * pi plus_equals (1/current cycle) * (subtract)
  * subtract = !subtract
* multiply pi by 4
* return pi
